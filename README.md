# 🎬 BiggPoppas Clip Bot

## 🚀 Overview

BiggPoppas Clip Bot is a fast, lightweight, and efficient tool designed to capture key moments from YouTube live streams in real time. With a simple `!clip` command triggered through Nightbot, the system automatically detects the currently live stream, calculates the exact timestamp, and generates a shareable clip link. This link is instantly delivered to a Discord server via webhook, making it effortless to preserve and share highlight moments as they happen.

Built with simplicity and performance in mind, this tool eliminates the need for manual clipping or editing, enabling streamers and communities to capture their best moments instantly without interrupting the live experience.

---

## ⚙️ Features

* 🎥 **Automatic Live Stream Detection** – Identifies active streams using YouTube Data API
* ⏱️ **Accurate Timestamp Clipping** – Captures precise moments from live streams
* 🔗 **Instant Clip Link Generation** – Creates shareable timestamp-based links
* 💬 **Discord Integration** – Sends clips directly to your server via webhook
* 🤖 **Nightbot Command Support** – Trigger clips using a simple `!clip` command
* ⚡ **Lightweight & Fast** – Minimal setup, maximum efficiency
* 🌍 **Free & Open for Everyone** – Designed for public use and customization

---

## 🛠️ Tech Stack

* **Backend:** Node.js + Express
* **API:** YouTube Data API v3
* **Integration:** Discord Webhooks
* **Automation:** Nightbot

---

## 📦 Usage

Once deployed and configured:

1. Start your YouTube live stream
2. Type `!clip` in chat
3. The bot automatically:

   * Detects the live stream
   * Calculates the timestamp
   * Generates a clip link
   * Sends it to Discord
